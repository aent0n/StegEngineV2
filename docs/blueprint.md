# **App Name**: Hideaway

## Core Features:

- File Import: Import various file types (images, text, PDFs, audio, video) to serve as the carrier for hidden messages.
- Algorithm Selection: Select from a suite of steganography algorithms to encode messages.
- Message Embedding: Hide text messages inside carrier files.
- Export: Download carrier files containing hidden messages.
- Algorithm Advisor: A generative AI tool can suggest the optimal algorithm based on the file type and message to be hidden, for better security.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to evoke feelings of mystery and security, which aligns with the app's function of hiding data.
- Background color: Very light indigo (#E8EAF6) for a clean, unobtrusive backdrop that keeps the focus on the content.
- Accent color: Complementary violet (#7986CB), to draw attention to key interactive elements without being distracting.
- Clean and modern sans-serif fonts for easy readability.
- Simple, monochromatic icons representing file types and actions.
- A minimalist, intuitive layout that is easy to navigate even for first-time users.
- Subtle animations to acknowledge user interactions.
